# Expense Tracker

A Java console-based expense tracking application.

## Features

- Add Expense
- View Expenses
- Update Expense
- Delete Expense
- Calculate Total Spending
- CSV File Persistence
- Auto-generated IDs

## Technologies

- Java
- ArrayList
- File I/O
- CSV Storage

## How to Run

javac Main.java Expense.java ExpenseManager.java

java Main
